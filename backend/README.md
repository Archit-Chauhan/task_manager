# WorkSync Pro 

WorkSync Pro Backend is a production-grade REST API built using Node.js, Express, and MongoDB.  
It handles authentication, role-based access control, shift scheduling, shift swap workflows,
task management, audit logging, and admin analytics.

---

## 🚀 Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- bcrypt (password hashing)
- Socket.IO (optional real-time updates)
- REST API architecture

---

## 👥 Roles Supported

- **ADMIN** – Full access, audit logs, analytics
- **MANAGER** – Shift & task management, approvals
- **EMPLOYEE** – View shifts, request swaps, manage tasks
